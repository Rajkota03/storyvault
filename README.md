# Story Vault v2 🛡️

**Story Vault** is a high-concept project management and brainstorming tool for screenwriters and storytellers. Designed with an "Industrial Dark" aesthetic, it prioritizes your core story "hook" (the Logline) while keeping technical logic securely tucked away in collapsible vault logs.

## ✨ Features

-   **Singular Focus**: High-Concept Logline is always visible at the top.
-   **Hybrid Layout**: Collaborative sections for Structure, Fusion Sources, Logic, and Architecture.
-   **Smart Parsing**: Paste your raw brainstorms; the vault automatically categorizes them using Step-based markers.
-   **Cloud Synced**: Powered by Supabase for real-time persistence and multi-device access.
-   **Industrial Aesthetics**: Premium dark theme with pulsing glow indicators and smooth transitions.

## 🚀 Getting Started

Simply open `index.html` in any modern browser. The application is a self-contained static site with external Supabase integration.

## 🛠️ Technology

-   **Frontend**: Vanilla HTML5, CSS3, JavaScript.
-   **Backend**: Supabase (PostgreSQL, RLS).
-   **Typography**: Inter & Playfair Display.
